They will update as you move between slides. All **shortcuts work** even when the presenter notes window is focused. Presenter notes **support Markdeep** and can be **defined using three nested blockquotes**, like so: ```markdown >>> Don't just read what's *on* the slides out loud, ya loon. >>> Here's some additional information: >>> * Cats are cute. ``` In draft mode, presenter notes are shown below each slide. --- ## Printing to PDF You can **export a PDF version** of your slides by simply **printing it to a PDF file** using your browser's built-in tools ‚Äì in my experience, this **works best in Chrome** as it respect the `@page` CSS selector properly. In the print modal, set "Margins" to "None" and make sure to keep the "Background graphics" option enabled. --- # Details --- ## Options There are TODO options: * `aspectRatio` defines the aspect ratio of your slides. ``` ``` Make sure to define your options *before* loading Markdeep. --- ## Tips I recommend using only local resources (images, fonts, a local copy of mathjax) unless you're absolutely sure you'll have internet connectivity where you'll be holding your presentation. * To download fonts from Google fonts, use https://google-webfonts-helper.herokuapp.com/fonts. * A guide on running MathJax locally can be found at http://docs.mathjax.org/en/latest/installation.html. Note that you'll have to add `detectMath: false` to your Markdeep config then. This has the added benefit of being able to configure MathJax to your liking. --- ## Themeing If your want to make your own theme, here are some helpful tips: * Make sure that all font sizes, margins, paddings, widths and heights are defined in terms of `rem` or `em`. * This enables your elements to scale properly depending on screen size. --- # Contributing Please file an issue or send a pull request. --- ## Terminology When going fullscreen, you switch from **draft mode** into **presentation mode**. --- ## Known issues * When resizing the window in Safari in draft mode, the font size is not scaled accordingly.
